# 最简实现流程

## 前端（`front/`）

```bash
cd front
npm install
npm run dev
```

```bash
npm run build
npm run preview
```

## 后端（`backend/`，Conda）

```bash
cd backend
conda env create -f environment.yml          # 仅首次；之后依赖变更用下面一行
# conda env update -f environment.yml --prune
conda activate uniai-video-backend
alembic upgrade head                         # 首次 / 拉代码有新迁移时执行
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

说明：`alembic`、`pytest`、`uvicorn` 都应在 **已激活** `uniai-video-backend` 的同一终端里执行。

## 仓库说明

- `docs/`：技术开发文档（主文档）
- `archive/`：原始交付包（zip + 解压后的方案/截图/参考代码）
- `front/`：前端工程
- `backend/`：API 服务（`/api/v1`，Swagger `/docs`）
- 根目录 `docker compose up -d --build`：起后端容器（上云可同款）

## 测试

后端（在 `backend` 目录，需已 `conda activate uniai-video-backend`）：

```bash
cd backend
alembic upgrade head
pytest
```

前端（在 `front` 目录）：

```bash
cd front
npm install
npm run test:run
```
