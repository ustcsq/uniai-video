## 阶段 0：地基 ✅

- [x] 配置扩展：`DATABASE_URL`、`REDIS_URL`、日志级别等
- [x] 数据库：SQLAlchemy 2 异步 + 会话；默认本地 SQLite（`data/dev.db`），可换 PostgreSQL
- [x] Alembic：迁移目录与首版空迁移
- [x] 统一错误体：`code` / `message` / `request_id`
- [x] 请求链路：`X-Request-ID` 中间件 + 基础日志
- [x] Redis：可选连接；`GET /api/v1/health` 汇总 `database` / `redis` 状态
- [x] Docker Compose：`postgres` + `redis`（API 通过环境变量连接）
- [x] 单元测试：`backend/tests`（pytest + TestClient）、`front`（Vitest + Testing Library）
